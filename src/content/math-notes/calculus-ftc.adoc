= The Fundamental Theorem of Calculus
:stem: latexmath
:sectnums:

== Introduction

The *Fundamental Theorem of Calculus* (FTC) is arguably the most important result in all of mathematics.
It reveals a profound duality: *differentiation* and *integration*, which appear to be entirely different operations,
are in fact inverse processes.

Before the FTC, computing areas and computing rates of change were seen as separate
geometric problems. Newton and Leibniz — independently — discovered the bridge between them.

---

== Preliminaries

We need two definitions before stating the theorem.

=== Definite Integral

Let stem:[f] be continuous on stem:[[a, b\]].
The *definite integral* of stem:[f] from stem:[a] to stem:[b] is defined as the limit of Riemann sums:

[stem]
++++
\int_a^b f(x)\, dx = \lim_{n \to \infty} \sum_{i=1}^{n} f(x_i^*)\, \Delta x
++++

where stem:[\Delta x = \dfrac{b-a}{n}] and stem:[x_i^*] is a sample point in each subinterval.

=== The Accumulation Function

Given a continuous function stem:[f] on stem:[[a, b\]], define the *accumulation function*:

[stem]
++++
F(x) = \int_a^x f(t)\, dt, \qquad x \in [a, b]
++++

stem:[F(x)] measures the signed area under the curve stem:[f] from stem:[a] up to the point stem:[x].
Notice that stem:[F(a) = 0] and stem:[F(b) = \int_a^b f(t)\, dt].

---

== Part I — Differentiation of an Integral

This is the heart of the FTC.

====
*Theorem (FTC Part I).* +
If stem:[f] is continuous on an open interval containing stem:[a], and

[stem]
++++
F(x) = \int_a^x f(t)\, dt
++++

then stem:[F] is differentiable and stem:[F'(x) = f(x)] for every stem:[x] in that interval.
====

In plain language: *the derivative of the area-accumulation function gives back the original function.*

=== Proof Sketch

We compute stem:[F'(x)] directly from the definition of the derivative:

[stem]
++++
F'(x) = \lim_{h \to 0} \frac{F(x+h) - F(x)}{h}
++++

Now stem:[F(x+h) - F(x) = \displaystyle\int_a^{x+h} f(t)\,dt - \int_a^x f(t)\,dt = \int_x^{x+h} f(t)\,dt].

Since stem:[f] is continuous on stem:[[x, x+h\]], the *Mean Value Theorem for Integrals* guarantees a point
stem:[c_h \in [x, x+h\]] such that:

[stem]
++++
\int_x^{x+h} f(t)\, dt = f(c_h) \cdot h
++++

Therefore:

[stem]
++++
F'(x) = \lim_{h \to 0} \frac{f(c_h) \cdot h}{h} = \lim_{h \to 0} f(c_h)
++++

As stem:[h \to 0], we have stem:[c_h \to x], and by continuity of stem:[f]:

[stem]
++++
F'(x) = f(x) \qquad \blacksquare
++++

== Part II — The Evaluation Theorem

This part gives us a practical method to compute definite integrals exactly — without Riemann sums.

====
*Theorem (FTC Part II).* +
If stem:[f] is continuous on stem:[[a, b\]] and stem:[G] is *any* antiderivative of stem:[f]
(i.e., stem:[G' = f]), then:

[stem]
++++
\int_a^b f(x)\, dx = G(b) - G(a)
++++
====

We write stem:[G(b) - G(a)] as stem:[\big[G(x)\big]_a^b] for compactness.

=== Proof

By Part I, stem:[F(x) = \int_a^x f(t)\,dt] is an antiderivative of stem:[f].
Any other antiderivative stem:[G] satisfies stem:[G(x) = F(x) + C] for some constant stem:[C].

Therefore:

[stem]
++++
G(b) - G(a) = \bigl[F(b) + C\bigr] - \bigl[F(a) + C\bigr] = F(b) - F(a)
++++

Since stem:[F(a) = 0] and stem:[F(b) = \int_a^b f(t)\,dt]:

[stem]
++++
G(b) - G(a) = \int_a^b f(x)\, dx \qquad \blacksquare
++++

=== A Worked Example

Compute stem:[\displaystyle\int_1^3 x^2\, dx].

An antiderivative of stem:[x^2] is stem:[G(x) = \dfrac{x^3}{3}]. By FTC II:

[stem]
++++
\int_1^3 x^2\, dx = \left[\frac{x^3}{3}\right]_1^3 = \frac{27}{3} - \frac{1}{3} = \frac{26}{3}
++++

== Part III — Consequences and Applications

=== Chain Rule with Integrals

When the upper limit is a function of stem:[x], apply the chain rule:

[stem]
++++
\frac{d}{dx} \int_a^{u(x)} f(t)\, dt = f\!\bigl(u(x)\bigr) \cdot u'(x)
++++

*Example:* Let stem:[H(x) = \displaystyle\int_1^{x^3} \sin(t^2)\, dt]. Then:

[stem]
++++
H'(x) = \sin\!\left(x^6\right) \cdot 3x^2
++++

=== Net Change Theorem

The integral of a rate of change gives the *net change*:

[stem]
++++
\int_a^b F'(x)\, dx = F(b) - F(a)
++++

For example, integrating velocity gives displacement:

[stem]
++++
\int_{t_1}^{t_2} v(t)\, dt = s(t_2) - s(t_1)
++++

=== Area Between Two Curves

If stem:[f(x) \geq g(x)] on stem:[[a, b\]], the area between the curves is:

[stem]
++++
A = \int_a^b \bigl[f(x) - g(x)\bigr]\, dx
++++

== Practice Problems

. Evaluate stem:[\displaystyle\int_0^{\pi} \sin(x)\, dx] using FTC II.
. Find stem:[F'(x)] where stem:[F(x) = \displaystyle\int_1^{x^2} \sqrt{1 + t^3}\, dt].
. Verify stem:[G(x) = \frac{2}{3}x^{3/2}] is an antiderivative of stem:[f(x) = \sqrt{x}] on stem:[[0, 4\]], then compute stem:[\displaystyle\int_0^4 \sqrt{x}\, dx].
. A particle moves with velocity stem:[v(t) = t^2 - 4t + 3] m/s. Find the *displacement* from stem:[t=0] to stem:[t=4].

NOTE: For Problem 4, displacement = stem:[\displaystyle\int_0^4 v(t)\, dt].
This differs from *total distance* = stem:[\displaystyle\int_0^4 |v(t)|\, dt].

== Summary

[cols="1,3", options="header"]
|===
| Result | Statement

| *FTC Part I*
| stem:[\dfrac{d}{dx}\displaystyle\int_a^x f(t)\,dt = f(x)]

| *FTC Part II*
| stem:[\displaystyle\int_a^b f(x)\,dx = G(b) - G(a)] where stem:[G' = f]

| *Net Change*
| stem:[\displaystyle\int_a^b F'(x)\,dx = F(b) - F(a)]

| *Chain Rule form*
| stem:[\dfrac{d}{dx}\displaystyle\int_a^{u(x)} f(t)\,dt = f(u(x))\cdot u'(x)]
|===
